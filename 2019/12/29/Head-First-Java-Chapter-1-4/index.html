Chapter 2

When child class extends super class

  • all method and attributes will be inheritance from super except private method and attributes.
  • child class can override method in super class

Chapter 3

floating-point number default type is double.

Reference

  • obj name in Java is reference only
  • obj name can be reuse to reference different obj of the same type
  • when an obj do not have any reference to access, it will be clean by the garbage collector

Chapter 4

  • When passing obj to method, we are passing a copy of the reference to the method, not the obj itself
  • The return value of a method must compatible with the declared type

Encapsulation

  • set the attributes to private and create setter and getter
  • -> ensure the data integrity

Default value of instace variable (obj attributes)

  • int/floting point number -> 0/0.0
  • bollean -> false
  • obj reference -> null

Local variable (variable inside method) do not have default value

  • intialization must be done before use
