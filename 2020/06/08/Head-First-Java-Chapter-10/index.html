Static method

  • static method do not rely on obj instance

  • class name is used to call the method

  • if a class containing only static method, we set the constructor to private to prevent creating instance obj from the class

  • static method cannot use the non static variable inside the class

  • see exp. below

  • static method cannot call the non static method

  • see exp. below

Static Variable

  • Obj of the same class maintain the same static variable (only one copy)
  • see exp. below
  • each duck has its own size, but all ducks maintain the same duckCount

Static final (Constant)

  • eg Math.PI
  • usually use of Capital letter and _ for naming

Static initializer

  • only run once (when the class is loaded by the JVM)
  • construct can run many times as long as obj is created

final

  • final on non static variable means non changable
  • final on method means cannot override
  • final on class means cannot inherit
  • all method inside final class are final method
