More about polymorphism

Be careful to the diff between Abstract class, Abstract method, interface.

Abstract class

  • Some classes are not suitable to create a instance eg. Animal, because we dont know what should a animal obj look like.
  • Abstract class help to prevent it being created as a instance obj.
  • The first concrete class must implement all the abstract methods in the class.
  • Abstract class have abstract method and non abstract method
  • see the exp below

Abstract methood

  • class contain any abstract method should declare as abstract class
  • ie. Non Abstract class CANNOT have Abstract method

Object class

  • all class extends object class
  • exp. method in obj class

    .equals() and ==

  • both compare if two refferece relate to the same obj
  • .equals() will usually be override to compare actual meaning of equality base on the obj.

interface

  • all methods are abstract
  • think interface as a contract, any class implement the interface are granteed to implenment the abstract methods
  • class can implement multiple interface
  • but can only extends from one class
